The frontend application for viewing analytics, managing sites, and generating tracking codes.

## Features
-   **Site Management**: Add and manage multiple domains.
-   **Analytics Visuals**: Real-time charts for views and unique visitors.
-   **Privacy-First**: No cookie handling or third-party trackers.

## Tech Stack
-   **Framework**: Next.js 15
-   **Styling**: Tailwind CSS
-   **Icons**: Lucide React
-   **Charts**: Recharts

## Environment Variables
Requires `NEXT_PUBLIC_API_URL` pointing to your Stat18ion API.

## Build and Run
Part of the monorepo. Use `npm run dev` at the root to run alongside the API.
